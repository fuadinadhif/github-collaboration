# Setup

## Initial Setup

1. Buat GitHub repository (repo)
2. Hubungkan git repo dengan local repo

## Git Commands

1. git branch: Melihat jumlah branch yang kita punya
2. git status: Melihat status branch yang sedang aktif
3. git checkout -b [BRANCH_NAME]: Membuat branch baru dan pindah ke branch tersebut
4. git checkout [BRANCH_NAME]: Pindah ke branch yang sudah dibuat sebelumnya tanpa membuat branch baru

## Update Flow

1. Buat feature branch (feat-1/auth)
2. Tulis/perbaiki code sampai tuntas di dalam feature branch tersebut (feat-1/auth)
3. Setelah selesai melakukan coding, push branch tersebut ke GitHub (git push origin feat-1/auth)
4. Buat pull request ke branch development
5. Beri notifikasi ke reviewer untuk mereview pull request kita
6. Kalau DIREJECT, perbaiki dan kembali ke poin 2 - 5
7. Kalau DIAPPROVE, sudah selesai.
