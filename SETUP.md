# Setup

## Initial Setup

1. Buat GitHub repository (repo)
2. Hubungkan git repo dengan local repo

## Git Commands

1. git branch: Melihat jumlah branch yang kita punya
2. git status: Melihat status branch yang sedang aktif
3. git checkout -b [BRANCH_NAME]: Membuat branch baru dan pindah ke branch tersebut
4. git checkout [BRANCH_NAME]: Pindah ke branch yang sudah dibuat sebelumnya tanpa membuat branch baru
